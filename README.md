# CaseySkillsState2026

For Program #1, usage would be formatted like this: ./Fibonacci-gen [-c|--count] 20 –one-line –last-only –numbering –help

For Program #2, basic usage would look like: python3 Program2weather_forecast.py <zip_code>
There are also command line options, such as: --output, --open, --help

For Program #3, I didn't utilize a GUI as the Mac that I borrowed was having issues with tkinter. Instead, I chose to stick to CLI to still utilize the operations of the code in tandem with the database. 

To add something as a example: python3 cash_register_cli.py add --sku A1 --description "Soda" --unit-cost 1.00 --sale-price 1.50 --quantity 20

To show inventory, use "list", to show summary, "summary".

Example usage of how sales are listed: python3 cash_register_cli.py sale --sku A1 --quantity 3 and to show what sales are made on a certain date: python3 cash_register_cli.py report --date <YYYY-MM-DD.
